b.State and Explain Final Algorithm¶

Multi Layer Perceptron Model

The multilayer perceptron model is a type of neural networking model that matches relevant inputs with related outputs. It is comprised of numerous layers of nodes. The input node is different from the other nodes and can be thought of as neuron, or processing core, with a specific activation function.. The multilayer perceptron model is capable of distinguishing data that is not separable in a linear way. The multilayer perceptron model uses the uses backpropagation algorithm that can be seen in a standard neural network. Multilayer perceptron’s can be found as part of the standard analysis algorithm for pattern recognition processes.

A perceptron has one or more inputs, a bias, an activation function, and a single output. In this cancer detection model, The perceptron receives 9 inputs, multiplies them by some weight, and then passes them into an activation function to produce an output. An activation function is a simple function that changes inputs into outputs that have a certain range. The activation used in this model is known as “ReLu” or Rectified Linear Unit. ReLu is a non-linear activation function that is used in multi-layer neural networks or deep neural networks. This function can be represented as:

The output of ReLu is the maximum value between zero and the input value.An output is equal to zero when the input value is negative and the input value when the input is positive.

Bias is also added to the perceptron, which prevents all inputs from becoming equal to zero.

There are 4 hidden layers of nodes, and 2 classes categorized as “2” for a benign cancer and “4” for a “malignant” cancer. The model took a maximum of 10000 iterations, with a train size of 0.8 and a test size of 0.2 to run the model on.

The final algorithm of the multi layer perceptron neural network model showed the following results:

Accuracy: Accuracy is the ratio of correct predictions to total predictions made. In MLP model, accuracy was 97%.

Precision: Precision is how accurate your model is . In MLP model, 97% of Class 2 benign cancers were correctly predicted, and 98% of Class 4 malignant cancers were correctly predicted. Overall precision is 97%.

Recall: Recall is the ratio of correctly predicted positive observations to all observations in actual class. In MLP model, Class 2 benign cancer can be correctly predicted 99% of the time. Class 4 malignant cancers can be correctly predicted 94% of the time. Overall recall is 96%

F1 score: F1 Score is the weighted average of Precision and Recall. In MLP model case, 97% is the F1 score. The confusion matrix that MLP neural network misclassified four cancer tumors out of 137 tumor test data the MLP neural network was predicted on.

Classification and Regression Trees or CART

CART for short refers to Decision Tree algorithms that can be used for classification or regression predictive modeling. A node represents a single input variable (X) and a split point on that variable. Leaf nodes (also called terminal nodes) of the tree contain an output variable (y) which is used to make a prediction. In the decision tree cancer model, a train size of 80%,test size of 20% was used.

The final algorithm of CART Decision tree model showed the following results:

Accuracy: Accuracy is the ratio of correct predictions to total predictions made. In CART model, accuracy was 96%.

Precision: Precision is how accurate your model is. In CART model, 96% of Class 2 benign cancers were correctly predicted, and 98% of Class 4 malignant cancers were correctly predicted. Overall precision is 97%.

Recall: Recall is the ratio of correctly predicted positive observations to all observations in actual class. In CART model, Class 2 benign cancer can be correctly predicted 99% of the time. Class 4 malignant cancers can be correctly predicted 92% of the time. Overall recall is 95%

F1 score: F1 Score is the weighted average of Precision and Recall. In CART model case, 96% is the F1 score. The confusion matrix that CART Decision tree misclassified 5 cancer tumors out of 137 tumor test data decision tree model was predicted on.

3.Identify 3 ways to Improve Performance of Model¶

1.Feature Selection-removing features in order to improve model performance. feature selection can be done on a neural network through creating a decision tree model. Decision trees are a separate type of machine learning model. Decision trees are useful as they can calculate and rank the most important featuresbased on various statistical criteria. We can first fit the data using the decision tree, and then use the output from the decision tree to remove features that are deemed unimportant, before providing the reduced dataset to our neural network.

2.Regularization - is a technique which helps to reduce overfitting in a neural network. When we add regularization to network, we add a new regularization term, and the loss function is modified.Dropout regularization is another regularization technique in which certain neurons are dropped along with their connections present in the neural network. Dropout regularization prohibits the neurons from relying only on certain features, and weights are spread out amongst all the inputs. The network uses all the input features to determine the output and does not rely on any single one, thus improving the neural network model.

Ensemble of Algorithms-If individual neural networks are not as accurate as benchmark that is ideal, method of emsemble is when various neural networks are combined to increase predictive power. Different neural network architectures can be used, and train the various neural networks on on different parts of the data and ensemble them and use their collective predictive power to get high accuracy on test data.
